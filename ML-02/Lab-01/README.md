## LAB1: Dataset Exploration

| ขั้นตอนการสำรวจข้อมูล | คำอธิบายรายละเอียด |
| :--- | :--- |
| **Load Dataset** | การโหลดหรือดึงชุดข้อมูลเข้ามาในโปรแกรมเพื่อเริ่มใช้งาน |
| **Display Shape** | การดูขนาดของข้อมูล (บอกจำนวนแถวและจำนวนคอลัมน์) |
| **Display Data Types** | การดูประเภทของข้อมูลในแต่ละคอลัมน์ (เช่น เป็นตัวเลข, ข้อความ, หรือวันที่) |
| **Display Summary Statistics** | การดูค่าสถิติเบื้องต้น (เช่น ค่าเฉลี่ย ค่ามากสุด ค่าต่ำสุด เพื่อดูการกระจายตัวของข้อมูล) |
| **Display Missing Values** | การเช็กว่ามีข้อมูลสูญหาย (ช่องว่าง/ค่าว่าง) ตรงไหนบ้าง |
| **Display Duplicate Records** | การเช็กว่ามีข้อมูลที่ซ้ำซ้อนกันหรือไม่ |
| **Display Class Distribution** | การดูสัดส่วนหรือการกระจายตัวของข้อมูลในแต่ละกลุ่ม |
