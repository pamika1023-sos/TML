## LAB3: Data Cleaning

| :--- | :--- |
| **Missing Value Handling** | การจัดการกับค่าที่สูญหาย (เช่น การลบแถวนั้นทิ้ง หรือการเติมค่าอื่นทดแทนลงไป) |
| **Duplicate Removal** | การลบข้อมูลที่ซ้ำซ้อนกันออกไป เพื่อไม่ให้ข้อมูลซ้ำมาดึงประสิทธิภาพหรือทำให้ผลลัพธ์เพี้ยน |
| **Incorrect Data Correction** | การแก้ไขข้อมูลที่ผิดพลาดให้ถูกต้อง (เช่น แก้คำสะกดผิด หรือค่าที่หลุดกรอบความเป็นจริง) |
| **Data Type Conversion** | การแปลงประเภทข้อมูลให้ถูกต้องและเหมาะสมกับการใช้งาน (เช่น แปลงตัวเลขที่ถูกเก็บเป็นข้อความ ให้กลับมาเป็นตัวเลขจริง ๆ) |
| **Mean** | ค่าเฉลี่ยเลขคณิต |
| **Median** | ค่ามัธยฐาน (ค่าที่อยู่ตรงกลางเมื่อนำข้อมูลมาเรียงลำดับ) |
