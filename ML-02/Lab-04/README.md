## LAB4: Feature Engineering

| ขั้นตอนการเตรียมฟีเจอร์ | คำอธิบายรายละเอียด |
| :--- | :--- |
| **Label Encoding** | การแปลงข้อความให้เป็นตัวเลขเรียงลำดับไปเรื่อย ๆ เช่น [แดง, เขียว, น้ำเงิน] จะถูกแปลงเป็น [0, 1, 2] |
| **One-Hot Encoding** | การแปลงข้อความเป็นคอลัมน์ใหม่ (สร้างคอลัมน์แยกตามประเภทข้อมูล) แล้วใส่เลข 1 ในคอลัมน์ที่เป็นจริง และ 0 ในคอลัมน์อื่น ๆ เช่น สร้างคอลัมน์ เป็นสีแดง?, เป็นสีเขียว? แยกออกมา เพื่อไม่ให้โมเดลเข้าใจผิดว่าเลขที่มากกว่ามีค่ามากกว่า (นิยมใช้กับข้อมูลที่ไม่มีลำดับขั้นชัดเจน เช่น ยี่ห้อรถ, จังหวัด, เพศ) |
